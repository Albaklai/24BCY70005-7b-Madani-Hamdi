## 📌 Overview
A professional shopping cart application built using **React and Redux Toolkit**.  
It includes real-time state management, input validation, and data persistence.

---

## 🚀 Features
- Add products with validation  
- Remove items from cart  
- Update quantity in real-time  
- Automatic total calculation  
- Data stored using LocalStorage  
- Responsive design  
- Error handling and user feedback

## 🛠️ Technologies Used

- React Redux
